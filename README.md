# Student Performance System

## Project Overview

The **Student Performance System** is a console-based application developed using **Python and MySQL** to manage student academic records efficiently.

The system allows users to store, retrieve, update, and analyze student data such as marks, attendance, and overall performance.

---

## Technologies Used

* Python
* MySQL
* SQL Queries
* Command Line Interface (CLI)

---

## Project Features

The system includes the following functionalities:

* Add new student records
* View student details
* Update student information
* Delete student records
* Record student marks and attendance
* Search students by ID or name

---

## Database Design

The project uses a **MySQL database** to store student information securely.

Example data fields include:

* Student ID
* Student Name
* Course / Department
* Marks
* Attendance

---

## System Workflow

1. User selects an operation from the menu.
2. The system connects to the MySQL database.
3. Data is stored, retrieved, or updated using SQL queries.
4. Results are displayed in the terminal.

---

## Project Goal

The goal of this project is to demonstrate how **Python and SQL can be integrated** to build a simple yet effective **student management system**.
